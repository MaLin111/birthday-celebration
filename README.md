# 生日快乐页面 🎂

一个炫酷科技风的生日快乐页面，使用 Vue3 + Vite 构建。

## 特性

- 🎨 渐变色动态标题文字
- ✨ 浮动粒子背景效果
- 💫 霓虹发光效果
- 🔍 扫描线科技感
- 🎂 旋转蛋糕图标
- 🎆 点击按钮触发烟花效果
- 📱 响应式设计，支持移动端

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
